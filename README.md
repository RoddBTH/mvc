# Requirements
* PHP
* Composer
* npm
* Symfony CLI

# Getting started
* Clone the repo using

```
git@github.com:RoddBTH/mvc.git
```

* Start the asset watch using
```
npm run watch
```

* Then start the server using

```
symfony server:start
```
or

```
php -S localhost:8888 -t public
```
